# Installation

Here is how you can install this project locally.

## Prerequisites

- Node.js 18+
- pnpm (recommended)

## Steps

1. Clone the repository
2. Install dependencies:

```bash
pnpm install
```

3. Start the dev server:

```bash
pnpm dev
```

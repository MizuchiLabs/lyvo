Welcome to **Lyvo Docs**, a sleek, high-performance, and minimal documentation template built with **Astro**, **React**, and **Tailwind CSS**. It is designed to be as close to a "drop-in" solution as possible while still offering endless customizability through raw code.

## Why Lyvo Docs?

There are many documentation generators out there like Starlight, Docusaurus, and Nextra. Lyvo Docs takes a different approach: **Simplicity and Ownership**.

Instead of hiding everything behind complex configuration files and abstract core libraries, Lyvo Docs provides you with the raw source code for your layouts and components. You own the code. If you want to change how the sidebar works, you just open `Sidebar.tsx` and change it.

### Key Features

Here is a quick overview of what you get out of the box:

- **Astro Powered:** Blazing fast performance by shipping zero JavaScript to the client by default.
- **React Components:** Interactive islands powered by React when you need them.
- **Tailwind CSS:** Beautiful, utility-first styling with dark mode support.
- **Shadcn UI:** Pre-configured with the standard of modern UI components.
- **MDX Support:** Write content in Markdown or MDX and seamlessly import React components.
- **Full-text Search:** Fast, offline-capable search using [Pagefind](https://pagefind.app/).
- **Responsive Design:** Looks great on desktop, tablet, and mobile devices.
- **Custom MDX Components:** Built-in callouts, tabs, code blocks, file trees, and accordions.

Getting up and running with Lyvo Docs is incredibly straightforward. It uses standard web tooling like `pnpm` and `Astro`.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18 or higher)
- A package manager like **pnpm** (recommended), **npm**, or **yarn**

## Installation

To create a new project using Lyvo Docs, clone the repository and install the dependencies:

```bash showLineNumbers
# Clone the template
git clone https://github.com/mizuchilabs/lyvo.git my-docs
cd my-docs

# Install dependencies
pnpm install
```

## Running the Development Server

Once the dependencies are installed, you can start the local development server:

```bash
pnpm dev
```

The server will start at `http://localhost:4321/`. Open this URL in your browser to see your documentation site live. Any changes you make to the `.mdx` files will be reflected instantly!

## Building for Production

When you are ready to deploy your site, run the build command:

```bash
pnpm build
```

This will generate a highly optimized, static version of your site in the `dist` directory. The build process also automatically runs **Pagefind** to generate the search index. You can preview the production build using:

```bash
pnpm preview
```

# Getting Started

Welcome to your new documentation site! This site is built with Astro, Tailwind CSS, and styled to be clean and minimal.

## Features

- Fast and lightweight
- MDX support out of the box
- Beautiful typography via `@tailwindcss/typography`
- Simple and elegant design inspired by Fumadocs and Mintlify
